Lecture 10: Loops

Computers are exceptionally good at doing repetitive tasks quickly and without making mistakes. If you want to print "Hello" 1,000 times, you shouldn't write 1,000 `std.println()` statements. Instead, you use a Loop. Loops allow you to execute a block of code repeatedly until a specific condition is met.

1. The `while` Loop

A `while` loop continues to run as long as its boolean condition evaluates to `true`. Before every single iteration (repeat) of the loop, the program checks the condition. If it is true, it runs the block. If it is false, the loop ends, and the program moves on.

mut counter: int = 1

while counter <= 5:
    std.println(std.format("Count: {}", counter))
    counter += 1  # Very important! Don't forget to update the condition.
    
std.println("Loop finished!")

Infinite Loops

If you forget to update the variable inside the `while` loop (like omitting `counter += 1`), the condition will always be true. Your program will get stuck in an infinite loop and print forever until it crashes or you force-quit it. To forcefully stop a running Shank program in your terminal, press `Ctrl + C`.

2. The `for` Loop and Ranges

When you know exactly how many times you want a loop to run, a `for` loop is usually the better choice. In Shank, `for` loops iterate over a sequence of numbers or an array. We use the `in` keyword alongside a range `start..end` (where `end` is exclusive).

# This will loop 5 times, with 'i' taking values 0, 1, 2, 3, 4
for i in 0..5:
    std.println(std.format("For loop iteration: {}", i))

3. Loop Control: `break` and `continue`

Sometimes you need to modify the flow of a loop from within its block. Shank gives you two keywords for fine-grained control:

The `break` Keyword

The `break` keyword immediately smashes out of the entire loop. No more iterations will occur, even if the loop's condition is still true.

mut target: int = 0
while true:  # Looks like an infinite loop!
    target += 1
    if target == 3:
        std.println("Target reached! Breaking out.")
        break  # Escapes the loop completely

The `continue` Keyword

The `continue` keyword skips the rest of the current iteration and jumps straight back to the top of the loop to evaluate the condition and start the next iteration.

for num in 1..6:
    if num == 3:
        # Skip printing the number 3
        continue
    std.println(std.format("Number: {}", num))
# Output will be: 1, 2, 4, 5
