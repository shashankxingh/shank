Why Shank?

⚡

Blazing Fast

Compiles directly to optimized x86-64 machine code via NASM and GCC. No virtual machines. No interpreters.

🐍

Pythonic Syntax

Forget semicolons and curly braces. Shank uses clean, indentation-based syntax that feels like writing Python.

📦

Standalone Executables

Produce a single, native `.exe` file that you can share anywhere. The Drag-and-Drop compiler makes it effortless.
