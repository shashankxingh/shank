A compiled language that doesn't waste your time.

Shank compiles to native x86-64 machine code. Write clean, indentation-based syntax. Get a standalone .exe out.

Download v0.1 Read the docs
example.sk 
im str name = put("What's your name? ")
im int age = put("How old are you? ")

out "{name} is {age} years old."

when age >= 18:
    out "You can vote."
otherwise:
    out "Wait {18 - age} more years."

def factorial(n):
    when n <= 1:
        return 1
    return n * factorial(n - 1)

out factorial(10)

What it does

⚡

Compiles to native code

Shank generates x86-64 assembly via NASM, links with GCC. No VM, no interpreter, no runtime dependency.

📝

Clean syntax

Indentation-based blocks. No semicolons, no curly braces. Top-level execution — no main function required.

🔒

Static types, checked at compile time

Four types: int, float, str, bool. Type errors caught before your code runs.

📦

Single .exe output

Produces a standalone Windows executable. Send it to anyone — they don't need Shank installed.

Get started

$ shank run hello.sk
