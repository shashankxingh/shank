Data Types

Shank's four primitive types.

int

64-bit signed integer. Range: −9,223,372,036,854,775,808 to 9,223,372,036,854,775,807.

im int x = 42
im int negative = -17

float

64-bit double-precision IEEE 754 floating-point number.

im float pi = 3.14159
im float small = 0.001

str

Null-terminated string. Enclosed in double quotes. Supports string interpolation with {} delimiters — any expression inside braces is evaluated and its result is inserted into the string.

im str greeting = "Hello, World!"

Interpolation example:

im int x = 42
im str msg = "Value is {x}"

bool

Boolean type. Exactly two possible values: true and false.

im bool flag = true
im bool done = false
