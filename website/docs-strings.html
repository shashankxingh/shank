String Interpolation

Embedding expressions inside string literals.

Basic Interpolation

Curly braces {} inside a string literal embed the value of a variable. The value is converted to its string representation at runtime.

im str name = "Shashank"
im int age = 20
out "My name is {name} and I am {age} years old."

Output:

My name is Shashank and I am 20 years old.

Supported Types

Interpolation works with all four primitive types. Each is converted to a string representation automatically.

im int x = 42
out "Value: {x}"        # Value: 42

im float f = 3.14
out "Pi is {f}"        # Pi is 3.140000

im bool b = true
out "Active: {b}"      # Active: True

im str s = "hello"
out "Word: {s}"        # Word: hello

Multiple Interpolations

A single string can contain any number of {} expressions.

im str name = "Shashank"
im int age = 20
im float height = 1.75
out "{name} is {age} and height is {height}"

Output:

Shashank is 20 and height is 1.750000

Concatenation

String interpolation replaces traditional concatenation. There is no explicit string concatenation operator in v0.1 — use {} to combine strings and other values inline.

im str first = "Shank"
im str second = "Lang"
out "{first}{second}"  # ShankLang
Note: Expressions inside {} are evaluated at runtime. Only variables and literals are currently supported inside interpolation braces — not complex arithmetic expressions.
