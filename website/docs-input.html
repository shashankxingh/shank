Input

Reading typed user input from stdin.

The put() function

put() prints a prompt string to stdout, then reads a full line from standard input. The return type is determined by the type of the variable the result is assigned to.

im int age = put("Enter your age: ")

The prompt string is displayed without a trailing newline. After the user presses Enter, the input is parsed into the declared type and bound to the variable.

Reading different types

The target variable's type annotation controls how the input line is parsed.

# read an integer
im int age = put("Enter your age: ")

# read a string
im str name = put("Enter your name: ")

# read a float
im float height = put("Height in meters: ")

# read a bool
im bool agree = put("Agree? (true/false): ")

Invalid input handling

If the user enters a value that cannot be parsed into the target type, the variable receives a type-specific default. No runtime error is raised — the program continues with the default value.

TypeDefault on invalid input
int0
float0.0
boolfalse (only the literal "true" is accepted as true)
strThe raw input line, as-is
 
# user enters "abc" when an int is expected
im int x = put("Enter a number: ")
out x   # 0

Using with mut

put() works with mutable variables. The input value can be reassigned or modified after reading.

mut int x = put("Enter x: ")
x = x + 1
out x
Note: put() always reads a full line. There is no way to read a single character in v0.1.
