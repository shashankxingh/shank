Loops

Bounded iteration with repeat.

repeat ... as

Iterates a fixed number of times. The loop counter starts at 0.

repeat <count> as <variable>:
    <indented body>
repeat 5 as i:
    out i

Output: 0, 1, 2, 3, 4 (each on a new line).

Dynamic Count

The count can be any integer expression, not just a literal.

im int n = 3

repeat n * 2 as idx:
    out "iteration {idx}"

Prints iteration 0 through iteration 5.

Using the Loop Variable

The loop variable (i, idx, etc.) is available inside the body as an int.

mut int total = 0

repeat 10 as i:
    total = total + i

out total    # prints 45

Nesting Loops

Loops can be nested. Each needs its own variable name.

repeat 3 as i:
    repeat 3 as j:
        out "{i},{j}"
repeat is the only loop construct in Shank v0.1. There is no while or for-in loop.
