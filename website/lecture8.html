Lecture 8: Input & Output

Up until now, our programs have been entirely self-contained. They do calculations internally, but they don't talk to the user. A program isn't very interactive if it can't ask for data or show you the result. In this lecture, we explore how Shank handles Output natively.

Topic 1: The `out` Keyword

Most languages require importing complex libraries or using bulky function calls like System.out.println() or printf() just to print something. Shank keeps it incredibly simple by introducing a native keyword just for output: out.

Subtopic 1.1: Basic Output

To print text, numbers, or booleans to the console, simply use the out keyword followed by the value or variable.

out "Hello World!"  # Prints a string
out 12              # Prints an integer
out True            # Prints a boolean

Subtopic 1.2: Printing Variables

You can effortlessly print the contents of a variable by passing the variable name directly to the out keyword.

im int age = 18
im str name = "Shashank"

out name
out age

Topic 2: String Interpolation

Often, you will want to mix text with variables inside a single sentence. Instead of clumsily adding strings together, Shank supports native inline formatting similar to f-strings, right out of the box.

Subtopic 2.1: Dynamic Injection

By placing curly braces {} around a variable name inside a string, Shank will automatically inject that variable's value into the text when it is outputted!

im int age = 18
im str name = "Shashank"

# Injects 'age' and 'name' seamlessly into the output string
out "Hey {name}, your age is {age}!"

Topic 3: Input: Listening to the User

To get data from the keyboard, we use std.input(). This pauses the program and waits for the user to type something and press Enter. The typed data is ALWAYS returned as a str (String) type.

out "Enter your name:"
im str user_name = std.input()
out "Welcome, {user_name}!"
