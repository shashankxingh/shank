Lecture 7: Operators

We know how to store data using variables and data types. But a program that just stores data isn't very useful. We need to manipulate, calculate, and compare that data. This is where Operators come in. Operators are special symbols that perform specific operations on one, two, or more variables/values.

1. Arithmetic Operators

These are the standard mathematical operators you learned in grade school, used primarily with `int` and `float` data types.

  • Addition (`+`): Adds two values.
  • Subtraction (`-`): Subtracts the right value from the left.
  • Multiplication (`*`): Multiplies two values.
  • Division (`/`): Divides the left value by the right.
  • Modulo (`%`): Returns the remainder of a division. Excellent for checking if a number is even or odd!
  • Power (`**`): Raises the left value to the power of the right value.
let a: int = 10
let b: int = 3

let sum: int = a + b      # 13
let remainder: int = a % b  # 1 (Because 10 divided by 3 leaves a remainder of 1)
let squared: int = a ** 2   # 100

2. Assignment Operators

You already know the basic assignment operator `=`. It assigns the value on the right to the variable on the left. But Shank also supports compound assignment operators, which update a variable based on its current value. Note: These require the variable to be declared as `mut`!

  • += Add and assign (e.g., x += 5 is the same as x = x + 5)
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign

3. Comparison Operators

Comparison operators look at two values and return a `bool` (`true` or `false`) based on their relationship. These are crucial for conditions and loops.

  • Equal to (`==`): Note the double equals! Single `=` is for assignment. Double `==` asks a question: "Are these equal?"
  • Not Equal to (`!=`):
  • Greater than (`>`):
  • Less than (`<`):
  • Greater than or equal to (`>=`):
  • Less than or equal to (`<=`):
let age: int = 18
let can_vote: bool = age >= 18   # This evaluates to true

4. Logical Operators

Logical operators allow you to combine multiple boolean conditions. Shank uses the plain English words `and`, `or`, and `not`.

  • `and`: Returns true ONLY if BOTH sides are true.
  • `or`: Returns true if AT LEAST ONE side is true.
  • `not`: Flips a boolean. True becomes false, false becomes true.
let has_ticket: bool = true
let is_vip: bool = false

# They can enter if they have a ticket OR if they are a VIP
let can_enter: bool = has_ticket or is_vip  # true

5. Operator Precedence (Order of Operations)

Just like in standard math (PEMDAS/BODMAS), Shank processes operators in a specific order. Multiplication and Division happen before Addition and Subtraction. If you want to force an operation to happen first, use parentheses `()`.

let result1: int = 10 + 5 * 2    # Equals 20
let result2: int = (10 + 5) * 2  # Equals 30
