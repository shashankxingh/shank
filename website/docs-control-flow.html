Conditionals

Branching with when and otherwise.

when

Executes its body if the condition is true.

when <condition>:
    <indented body>
when x > 10:
    out "big number"

otherwise

The fallback block. Executes when the preceding when condition is false.

im int age = 15

when age >= 18:
    out "Adult"
otherwise:
    out "Minor"

Nesting

You can nest when/otherwise blocks inside each other.

when x > 100:
    out "huge"
otherwise:
    when x > 50:
        out "medium"
    otherwise:
        out "small"

Conditions

The condition must evaluate to a bool. You can use comparison operators (==, !=, <, >, <=, >=) and combine them with logical operators (and, or, not).

when score >= 90 and passed == true:
    out "Excellent"
Shank does not have elif in v0.1. Use nested when/otherwise blocks instead.
