Shank Documentation

Language reference for Shank v0.1 — a statically typed, compiled language with Python-like syntax.

Shank compiles .sk source files directly to x86-64 machine code via NASM and GCC. The output is a standalone Windows executable.

# compile and run
shank run myfile.sk

# compile to .exe only
shank build myfile.sk -o myfile.exe

# type-check without compiling
shank check myfile.sk

Programs execute top-level statements from top to bottom. No main function is required.

out "Hello, World!"
