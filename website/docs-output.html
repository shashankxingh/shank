Output

Printing values to standard output.

The out Keyword

out prints a value followed by a newline. It works with all four types.

out 42
out "hello"
out 3.14
out true

Output:

42
hello
3.14
True

Printing Expressions

out evaluates the expression before printing the result.

im int x = 10
out x * 2   // prints 20
out x > 5   // prints True

Printing Variables

Pass the variable name directly to out.

im str name = "Shashank"
out name   // prints Shashank

String Interpolation with out

out works with interpolated strings. Expressions inside {} are evaluated and inserted into the string.

im str name = "Shashank"
out "Hello, {name}!"   // prints Hello, Shashank!

See Strings for full interpolation syntax.
