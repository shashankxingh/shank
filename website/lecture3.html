Lecture 3: IDE Setup for Coding

You know the logic, you understand how the compiler works, and now it is time to actually write some code. While you could write Shank code in Windows Notepad, doing so would make your life miserable. Programmers use specialized software called an Integrated Development Environment (IDE) or an advanced code editor to write, manage, and debug their code.

1. Choosing an Editor: Visual Studio Code

For Shank, we highly recommend Visual Studio Code (VS Code). It is free, lightweight, incredibly fast, and has excellent extension support. It provides essential features like syntax highlighting, auto-completion, and integrated terminals.

Step-by-Step Installation:

  1. Go to the official VS Code website.
  2. Download the Windows installer.
  3. Run the installer and follow the standard prompts. Make sure to check the boxes that say "Add to PATH" and "Add 'Open with Code' action to Windows Explorer context menu".

2. Installing the Shank Tools

Because Shank is a custom compiled language, you need the Shank compiler installed on your system, as well as an assembler and linker.

Prerequisites: NASM and GCC

Shank compiles to NASM assembly, which is then linked using GCC.

  • NASM: Download and install the Netwide Assembler from nasm.us. Add the installation folder to your Windows System PATH.
  • GCC: The easiest way to get GCC on Windows is via MSYS2 or MinGW. Install it and ensure `gcc.exe` is in your PATH.

The Shank Compiler

Download the latest Shank compiler binary from the Downloads page. Extract it to a folder (e.g., C:\Shank\bin) and add that folder to your System PATH environment variable. Once added, you can open a terminal and type:

shank --version

If you see the version number printed, your setup is successful!

3. Setting up VS Code for Shank

To get syntax highlighting (so your keywords, strings, and functions show up in different colors), you should install the Shank Language Extension.

  1. Open VS Code.
  2. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window (or press `Ctrl+Shift+X`).
  3. Search for "Shank" (or install the `.vsix` extension file provided in your course materials manually).
  4. Install the extension and reload VS Code.

4. Writing and Running Your First File

Let's create a workspace. Best practice is to keep all your code organized in a dedicated folder, rather than scattering files across your Desktop.

  1. Create a folder called ShankProjects in your Documents.
  2. Right-click the folder and select "Open with Code".
  3. In VS Code, click the "New File" icon and name it hello.shnk. All Shank files must use the .shnk extension.
  4. Type the following code:
import std

fn main() -> int:
    std.print("IDE Setup Complete! Ready to code.")
    return 0

Running the Code

Open the integrated terminal in VS Code (`Ctrl + ~`). Make sure you are in the directory containing hello.shnk. Run the compiler:

shank build hello.shnk
.\hello.exe

You should see your success message printed to the console! You are now fully set up and ready to dive into the core features of the Shank language.
